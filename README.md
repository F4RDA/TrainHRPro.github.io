# TrainHRPro.github.io

TrainHR Pro respects your privacy. This Privacy Policy explains how we handle information when you use our mobile application.

  1. Information We Collect
  TrainHR Pro does not collect, store, or transmit any personal data.
  The app may ask you to manually enter the following information:
  Age
    Resting heart rate
    Maximum heart rate (optional)
    Training intensity preferences
    This information:
    Is entered voluntarily by the user
    Is processed only on the device
    Is not stored, not shared, and not transmitted anywhere
  2. How We Use Information
    The entered data is used solely to calculate heart rate training zones using the Karvonen method and display the results within the app.
  3. Data Storage
    All calculations happen locally on your device
    No data is saved on external servers
    No cloud storage is used
  4. Third-Party Services
    TrainHR Pro does not use:
    Analytics tools
    Advertising SDKs
    Third-party trackers
    External APIs
  5. Permissions
    TrainHR Pro does not require access to:
    Location
    Camera
    Microphone
    Contacts
    Sensors
    Health data
